2017 of mine

我的 2017, 年中总结

Posted by mslinzz on 2018-01-20

前言

2017, 忙的不亦乐乎.

感谢

工作年限的长短和工作环境确实能受益一个人, 影响一个人. 首先, 2017 年, 我非常感谢我所在的前端团队, 感谢我敬爱的 Leader, 感谢我组的小伙伴们, 很欣慰能和他们在一起快乐的编程.

成果

维护旧版官网商城 - 个人中心 - 订单流程 - 付款流程等 (Backbone+Chameleon)
维护第三方平台 - 账户中心 (Angular1+Gulp)
云端设备查找 (貌似现在还没有对外开放入口) (Angular1+Gulp)
初始云相册页面及框架搭建 (因优先级暂缓开发一年, 后由小组另外一位小伙伴重新接手并顺利上线) (Angular2+Webpack)
开发新版官网商城 (PC & Mobile) (Angular2+Webpack)
迁移旧版官网商城遗留页面 (Angular2+Webpack)

阅读书籍

<< Angular Js 权威教程 >>
<< 图解 HTTP >>
<< JavaScript 高级程序设计 >> (心碎, 断断续续读了整整一年的时间)

此外, 还大致浏览过

Angular JS 2 的官网教程及API, 电子版书籍 (60%)
微信小程序官网API (60%)
<< Web 前端开发最佳实战 >> (20%)

2018 技术提升

重温: << JavaScript 高级程序设计 >>

阅读: << JavaScript 设计模式与开发实战 >> (Flag: 春节假日读完)

继续: << Web 前端开发最佳实战 >> (20%)

阅读: << ES6 标准入门 >>

了解: Node 全站开发 (待思考)

最后的总结

我认为 2017 技术最大的提升是:

1.真正理解 组件式开发 的基本思想和理念;
2.熟练学会了 Web 开发 调试工具 和 ‘抓包工具’;
3.全新了解 MVVM模式 开发的好处;
4.感受到由gulp, webpack等搭建的自动化开发的便捷;
5.UI的小到 1像素 的抓狂.

2018

继续努力, 喜欢编程, 享受编程;
